# Portals IDE
Cool IDE for people who want to browse the web for code, installers and tutorials while coding simultaneously. 
AI optionally included as an option in the config but disabled as default.

# How to install
1. Run the `install.sh` script on Linux/macOS or `install.bat` on Windows
2. If you need (or want) to, run the `install_for_all.sh` script to install for all on Linux/macOS or `install_for_all.bat` for Windows. Requires running with admin on Windows and sudo on Linux/macOS.
3. If using AI, install Ollama, set up Ollama, start Ollama and edit in the config to say `AI_PROVIDER=ollama` and, in the net line, `RUN_AI=ollama run <your ai model>`
4. Configure to your needs.

# Why use this IDE?
The web browser and optional-in-config AI and stuff.
